# Context Keeper

> ⚠️ Placeholder copy — repo is private and the live demo is gated, so I couldn't auto-extract the real description. Edit the frontmatter and body below with the actual pitch.

A tool for keeping LLM coding agents grounded in your project across sessions. Instead of starting from zero every conversation, Context Keeper maintains a living record of architecture, decisions, and conventions that an agent can read at the start of any task.

## Why

Long coding sessions with AI assistants degrade as context windows fill up. Cross-session continuity is worse — the model forgets every architectural choice you made yesterday. Context Keeper acts as durable memory: structured, queryable, and small enough to inject into a system prompt.

- Persistent project memory across sessions
